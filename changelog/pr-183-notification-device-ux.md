# Notification-Geräteverwaltung und Installationshinweise

## Zusammenfassung

- Registrierte eigene Push-Geräte lassen sich ohne Seitenreload umbenennen.
- iOS/iPadOS verlangt vor der Push-Aktivierung die Installation auf dem Home-Bildschirm.
- Desktop- und Android-Browser kennzeichnen die Installation als optional.

## Geänderte Dateien

- Notification-Endpunkte, URLs, Template, JavaScript und Styles
- Pytest- und browserübergreifende Playwright-Regressionstests

## Tests

- Fokussierte Notification-Pytests
- Fokussierte PWA-Playwright-Tests in Chromium, Firefox und WebKit
