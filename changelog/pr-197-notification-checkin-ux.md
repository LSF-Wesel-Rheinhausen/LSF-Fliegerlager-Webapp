# Benachrichtigungs- und Check-in-Korrekturen

- Bereits registrierte Push-Geräte können ihre Benachrichtigungsauswahl nachträglich ändern.
- Check-in-Datumsgrenzen werden verständlich auf Deutsch angezeigt.
- Das mobile Check-in-Formular bleibt ohne verschachtelte Scrollbereiche vollständig bedienbar.
- Installierte PWAs laden die aktualisierten Check-in-Styles über eine neue Cache-Version.
