# Abendessen-Kalender im Verpflegungsbereich des Kiosks

- Der Abendessen-Kalender kann im Kiosk nun direkt über eine eigene Schaltfläche auf der Karte "Verpflegung buchen" geöffnet werden.
- Die Bezeichnung im Menü "Weitere Bereiche" wurde in "Abendessen (Kalender)" geändert, um die Abgrenzung zu Snacks und Frühstück zu verdeutlichen.
