## PWA-Installation und Benachrichtigungen

- Die Installation zeigt jetzt eine passende Anleitung für iOS, Android und Desktop-Browser.
- Im Central-Kiosk-Modus wird keine Installationsaktion mehr angeboten.
- Push-Geräte werden ohne Seitenreload registriert, aktualisiert und entfernt.
- Installierte Apps verwenden das Vereinslogo auch auf iPhone und iPad.
