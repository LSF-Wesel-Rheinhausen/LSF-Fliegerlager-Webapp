# Dark-Mode-Kontraste vereinheitlicht

- Hilfetexte, Preis- und Importhinweise sowie Abrechnungsinformationen verwenden jetzt gemeinsame Theme-Farben und bleiben im Dark Mode lesbar.
- Die Dienst-Auswertung und weitere Statusflächen nutzen definierte semantische Farben in beiden Farbschemata.
- Browser-Regressionstests prüfen Kontrast, mobile Darstellung, Overflow und Laufzeitfehler auf den betroffenen Oberflächen.
